Title

Interactive Coding Assistant: Applying Higher-Order Functions (HOFs)

Level

Level 0

Beginner

Overview

As an interactive coding assistant AI, you are tasked with guiding users through basic concepts of Higher-Order Functions (HOFs). You will design prompts that help users understand and apply HOFs to solve common programming tasks.

Detailed Explanation

Context

You are developing an AI coding assistant to help beginners understand and apply Higher-Order Functions (HOFs) effectively. Your goal is to create interactive prompts that guide users through simple coding exercises where they utilize HOFs to manipulate data.

Requirements

  1. Prompt Design:
    • Design three prompts that introduce different aspects of HOFs (e.g., map, filter, reduce).
    • Each prompt should focus on a specific use case where HOFs are beneficial in simplifying code and improving readability.
  2. Implementation Instructions:
    • Provide clear instructions on how users can implement each prompt using JavaScript.
    • Include sample input and expected output to demonstrate the functionality of HOFs.
  3. Edge Cases:
    • Consider edge cases such as empty arrays, undefined values, or unexpected data formats.
    • Implement error handling or validation techniques where appropriate to ensure robust functionality.

Example Prompts:

  1. Mapping Array Elements:
    • Prompt the user to input a list of numbers separated by spaces. Implement a function that uses the map HOF to double each number and display the transformed array."
    • Example Input:
      let str = "1 2 3 4"
      let res = doubleArray(str)
      console.log(res)
    • Expected Output: "[2, 4, 6, 8]"
  2. Filtering Data:
    • "Ask the user for a list of integers separated by commas. Create a function that uses the filter HOF to extract even numbers from the input and display them."
    • Example Input:
      let str = "1, 2, 3, 4, 5"
      let res = filterArray(str)
      console.log(res)
    • Expected Output: "[2, 4]"
  3. Reducing Arrays:
    • "Prompt the user for a series of numbers separated by semicolons. Implement a function that uses the reduce HOF to calculate the sum of the numbers and display the result."
    • Example Input:
      let str = "5; 10; 15"
      let res = totalArray(str)
      console.log(res)
    • Expected Output: "30"
