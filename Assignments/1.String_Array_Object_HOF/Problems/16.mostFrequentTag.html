Problem : Most Frequent Tag Identifier

Description:

Given a mixed array containing strings and objects (with a tags array), find and return the most frequently occurring tag.

Input:

const input = [
"react",
{ tags: ["js", "frontend", "react"] },
{ tags: ["frontend", "css"] },
"css",
{ tags: ["react", "js"] }
];
console.log(mostFrequentTag(input));

Expected Output:

"react"
