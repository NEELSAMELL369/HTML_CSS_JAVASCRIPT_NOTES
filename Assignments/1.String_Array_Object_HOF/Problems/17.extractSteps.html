Problem : Extract Actions and Step Counts from Text

Description:

Given a sentence containing actions and step counts in mixed case, return an object showing the total step count per action (case-insensitive).

Input:

const input = "Run 45 steps, then Jump 30 STEPS, finally run 25 Steps!";
console.log(extractSteps(input));

Expected Output:

{ run: 70, jump: 30 }

Use: split, toLowerCase, Number, includes, trim, etc.
