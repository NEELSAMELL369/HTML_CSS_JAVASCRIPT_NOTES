🌡️ Problem: Color Temperatures

📝 Description

You are given an array of temperatures. Write a function to classify each temperature into one of the following categories based on its value:

  • "green" for temperatures less than 20
  • "yellow" for temperatures from 20 to less than 35
  • "red" for temperatures 35 or above

The function should return an array of color labels corresponding to each temperature.

📥 Input Description

  • A single array of integers temps representing temperatures.

📤 Output Description

Return an array of strings with each string being either "green", "yellow", or "red", based on the rules above.

📌 Constraints

  • 1 ≤ temps.length ≤ 1000
  • -100 ≤ temps[i] ≤ 100

🧪 Sample Input

[15, 22, 38]

✅ Sample Output

["green", "yellow", "red"]

💡 Hint

Use conditional logic inside a map() function to check each value and assign the appropriate label.
