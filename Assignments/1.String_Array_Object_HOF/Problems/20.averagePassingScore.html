Problem : Chained .map → filter → reduce

Task

Create averagePassingScore(scores) that:

  1. maps each student record { name, marks } to its numeric marks,
  2. filters marks ≥ 40 (pass),
  3. reduces to the arithmetic mean (rounded to one decimal).

All steps must be chained in a single expression—no intermediate variables.

Example Input

let scores = [
{ name: "A", marks: 35 },
{ name: "B", marks: 65 },
{ name: "C", marks: 85 },
{ name: "D", marks: 42 }
]
let res = averagePassingScore(scores)
console.log(res)

​Expected Output

[ 64.0]
