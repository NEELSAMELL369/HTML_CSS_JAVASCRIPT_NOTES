Problem : Strings × Arrays × Objects × Destructuring

Task

Write parseInventory(csv) where csv is a newline-delimited string:

"Item,Category,Qty\nKeyboard,Electronics,12\nPen,Stationery,30"

Steps:

  1. Split the string into rows & columns.
  2. Return two results in an array via array-destructuring:
  3. A list of objects { item, category, qty } (with qty as number).
  4. The total quantity of all items.

Example Input

let csv = "Item,Category,Qty\nKeyboard,Electronics,12\nPen,Stationery,30"
let res = parseInventory(csv)
console.log(res)

​Expected Output

[ [
{ item: "Keyboard", category: "Electronics", qty: 12 },
{ item: "Pen", category: "Stationery", qty: 30 }
],
42
]
