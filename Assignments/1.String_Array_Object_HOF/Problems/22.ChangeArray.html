Change Array

Description

You are given an array A of size N.
You need to change all numbers in the array which are divisible by the smallest number in the array, to -1 and print the array.

Input Description

First line contains T, the number of test cases.
For each test case:

  • First line contains N - the size of the array
  • Second line contains N space-separated integers of array A

Constraints

  • 1 ≤ T ≤ 10
  • 1 ≤ N ≤ 100
  • 1 ≤ A[i] ≤ 1000

Output Description

For each test case, print the updated array in a new line.

Sample Input 1

function changeArray(arr) { 

 

} 

 

let arr = [3, 3, 2, 9, 7] 

let res = changeArray(arr) 

console.log(res)

Sample Output 1

3 3 -1 9 7
Hint:
For test case 1, the smallest number is 1. All elements are divisible by 1, so the entire array becomes -1.
For test case 2, the smallest number is 2. Only 2 is divisible by 2, so only that element is changed to -1.
