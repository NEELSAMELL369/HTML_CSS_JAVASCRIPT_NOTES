Move Zeros

Description

Given an array of N integers, with integers ranging from -10 to 10, move all zeros to the end of the array while maintaining the relative order of the non-zero elements.

Input Description

  • 1 ≤ N ≤ 100000
  • -10 ≤ Ai ≤ 10

Output Description

Output N integers, with the relative order of non-zero elements unchanged, and all zeros moved to the end.

Sample Input 1

let arr = [0 ,1 ,2 ,0]
let res = moveZeros(arr)
console.log(res)

Sample Output 1

1 2 0 0
