Description

You are given two buckets with capacities bucket1 and bucket2 (in liters), and a target volume target. Your goal is to measure exactly target liters using the two buckets.

You can perform the following operations:

  • Fill either bucket completely.
  • Empty either bucket.
  • Pour water from one bucket into the other until one is empty or the other is full.

Write a function to find the minimum number of steps required to measure exactly target liters of water.

Sample Input

const a = [3, 5];
const b = [3, 6];

const res = canPourIntoTwoBuckets(a, b);
console.log(res);

Sample Output

6

Explanation

Steps to get 4 liters using 3L and 5L buckets:

  1. Fill 5L bucket (5, 0)
  2. Pour from 5L into 3L (2, 3)
  3. Empty 3L (2, 0)
  4. Pour from 5L to 3L (0, 2)
  5. Fill 5L (5, 2)
  6. Pour from 5L to 3L (4, 3)
Total steps: 6
