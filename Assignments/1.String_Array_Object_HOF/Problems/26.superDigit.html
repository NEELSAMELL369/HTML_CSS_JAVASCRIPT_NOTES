Description

You are given two numbers n and k. The number p is created by concatenating the string n k times.
You need to find the super-digit of the integer p.

We define superDigit of an integer p using the following rules:

  • If p has only 1 digit, then superDigit(p) is p itself.
  • Otherwise, superDigit(p) is equal to superDigit(sum of the digits of p).
Read the sample input explanation for better understanding.
