🔡 Count Character Frequencies

Given an array of strings, write a function that counts how many times each character appears across all the strings. Return an object where the keys are individual characters and the values are their total counts.

🔍 Input Format:

An array of lowercase strings. Each string contains only alphabetic characters (a-z).

🎯 Output Format:

An object where each key is a character, and its value is the total number of times that character appears in the array.

💡 Example:

Input:
["abc", "bca", "cab"]

Output:
{ a: 3, b: 3, c: 3 }

✅ Constraints:

  • 1 ≤ arr.length ≤ 10⁴
  • Each string contains only lowercase letters.
  • Total combined string length ≤ 10⁵
