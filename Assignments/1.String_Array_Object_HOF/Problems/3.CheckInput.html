🧪 Check if Input is an Array

Write a function isArray(input) that determines whether the given input is an array or not. The function should return true if the input is an array, and false otherwise.

🔍 Input Format:

Any JavaScript value (array, object, number, string, etc.).

🎯 Output Format:

A boolean value: true if input is an array, false otherwise.

💡 Example:

Input: [1, 2, 3]
Output: true

Input: "hello"
Output: false

Input: { name: "John" }
Output: false

✅ Constraints:

  • No external libraries should be used.
  • Use JavaScript’s built-in methods to check the data type.
