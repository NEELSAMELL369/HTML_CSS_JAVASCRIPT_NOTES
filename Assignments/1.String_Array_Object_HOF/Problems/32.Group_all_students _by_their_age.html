👥 Group Users by Age

Given a list of user objects where each object contains a user (string) and an age (number), write a function that groups users by their age. The output should be an object where:

  • Each key is an age (as a number).
  • The corresponding value is an array of usernames who have that age.

🔍 Input Format:

An array of objects, each with a user and age property.

🎯 Output Format:

An object where keys are ages and values are arrays of usernames.

💡 Example:

Input:
function groupUsersByAge(arr) {
    
}
const arr = [
    { user: "Alice", age: 25 },
    { user: "Bob", age: 30 },
    { user: "Charlie", age: 25 },
    { user: "David", age: 30 },
    { user: "Eve", age: 35 }
];

console.log(groupUsersByAge(arr));

Output:
{
  25: ["Alice", "Charlie"],
  30: ["Bob", "David"],
  35: ["Eve"]
}

✅ Constraints:

  • 1 ≤ number of users ≤ 10⁴
  • Usernames are non-empty strings.
  • Age is a positive integer.
