Title

Calculate Total Expenses

Level

Level 0

Beginner

Overview

In this task, you will calculate the total expenses incurred over a period using an array of expense items.

Detailed Explanation

Setup

You have an array of expense items, where each item is an object containing name and amount.

Task Requirements

  1. Calculate Total Expenses:
    • Sum up the amount values from all expense items to get the total expenses.
  2. Output Format:
    • Return a single number that represents the total expenses incurred.

Example

// Input
const expenses = [
{ name: "Food", amount: 50 },
{ name: "Transportation", amount: 30 },
{ name: "Internet", amount: 45 },
{ name: "Groceries", amount: 80 }
];
const totalExpenses = calculateTotalExpenses(expenses);
console.log(totalExpenses);

// Output should be: 205
