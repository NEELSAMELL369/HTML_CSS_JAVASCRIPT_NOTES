Employees in Department: Define a function employeesInDepartment that takes an array of employee objects, employees, and a string department as parameters. Filter out the employees who belong to the given department and return an array of their names.

//Example Invocation:

const employees = [
{ name: "John Doe", age: 30, department: "HR", salary: 50000 },
{ name: "Jane Smith", age: 28, department: "Finance", salary: 60000 },
{ name: "Alex Johnson", age: 35, department: "IT", salary: 70000 },
{ name: "Maria Lopez", age: 29, department: "Finance", salary: 65000 },
];
console.log(employeesInDepartment(employees, "Finance"));

// Output: ['Jane Smith', 'Maria Lopez']
