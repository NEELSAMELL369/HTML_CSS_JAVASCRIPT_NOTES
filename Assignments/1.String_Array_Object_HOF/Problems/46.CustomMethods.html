Title

Custom Array Methods Assignment

Level

Level 2

Overview

In this assignment, you will implement custom methods for arrays in JavaScript. These methods mimic functionalities of built-in methods like map, filter, reduce, and include your own unique methods such as customSlice, customSplice, customPush, and customPop. Each method should be designed to operate similarly to its built-in counterpart but with unique features or constraints.

Detailed Explanation

Setup

Begin by creating a JavaScript file to write your code. You can use any text editor or IDE of your choice for editing.

Running Your Code

You can run your JavaScript code in any modern browser's developer console or using Node.js.

Problem Statement

You are required to implement a series of custom methods for arrays based on the following specifications:

Custom Higher-Order Functions

  1. customMap
    • Implement a custom map function that takes a callback function as an argument.
    • Apply the callback function to each element in the array and return a new array with the results.
  2. customFilter
    • Implement a custom filter function that takes a predicate function as an argument.
    • Apply the predicate function to each element in the array and return a new array containing only the elements that satisfy the predicate.
  3. customReduce
    • Implement a custom reduce function that takes a reducer function and an initial value as arguments.
    • Apply the reducer function to each element in the array, combining them into a single value.

Example Usage

const arr = [1, 2, 3, 4, 5];
// Using custom methods
console.log(arr.customSlice(1, 3)); // Output: [2, 3]
console.log(arr.customSplice(2, 0, 6)); // Output: [1, 2, 6, 3, 4, 5]
console.log(arr.customPush(6, 7)); // Output: 7 (new length of the array)
console.log(arr.customPop()); // Output: 5 (removed element)
// Using custom higher-order functions
console.log(arr.customMap(x => x * 2)); // Output: [2, 4, 6, 8, 10]
console.log(arr.customFilter(x => x % 2 === 0)); // Output: [2, 4]
console.log(arr.customReduce((acc, cur) => acc + cur, 0)); // Output: 15

Guidelines

  • Ensure that your custom methods behave similarly to their built-in counterparts.
  • Test each method with various inputs to verify correctness.
  • Document your code and explain any deviations or unique features compared to built-in methods.
