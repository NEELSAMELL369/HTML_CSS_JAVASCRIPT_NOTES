Problem Statement:

You are given an array of objects where each object represents a task. Each task contains the following properties:

  • description: a string describing the task
  • completed: a boolean indicating whether the task is completed or not

Your task is to:

  1. Sort the tasks by their completion status, showing incomplete tasks first.
  2. Filter out the tasks that are already completed.
  3. Display the sorted and filtered list of tasks on the web page.

Requirements:

  • A button that triggers the sorting and filtering functionality.
  • Display each task's description and completion status on the page.
