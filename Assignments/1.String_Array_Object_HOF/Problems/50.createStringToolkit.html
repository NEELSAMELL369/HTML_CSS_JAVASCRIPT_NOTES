Problem : Function Returning an Object with Rest-Aware Methods

Task

Implement createStringToolkit(separator = " ") that returns an object with three methods, all defined as arrow functions and all using rest (...) parameters:

  1. join(...strings) → concatenates with the chosen separator
  2. titleCase(...strings) → calls .join, then converts each word’s first letter to uppercase
  3. stats(...strings) → returns an object { count, longest } where count is total strings joined and longest is the longest individual string supplied

Do not use class syntax or traditional functions.

Example Input

const tk = createStringToolkit("-");
console.log(tk.join("hello", "world"));
console.log(tk.titleCase("the", "quick", "fox"));
console.log(tk.stats("a", "ab", "abc"));

​Expected Output

"The-Quick-Fox"
{ count: 3, longest: "abc" }
