🧩 Problem: Are Anagrams?

📝 Description

You are given two strings. Your task is to determine whether the two strings are anagrams of each other.

Two strings are called anagrams if they contain the same characters, but possibly in a different order. The comparison should be case-insensitive. Ignore any non-alphabetic characters (like punctuation or spaces).

🧾 Input Description

  • First line: a string str1
  • Second line: a string str2

📤 Output Description

Print "true" if the strings are anagrams, otherwise print "false".

🔒 Constraints

  • 1 ≤ length(str1), length(str2) ≤ 10⁵
  • The strings may contain letters, spaces, and punctuation.
