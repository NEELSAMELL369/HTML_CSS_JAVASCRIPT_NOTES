🧠 Problem: Happy Number

🔍 Description

You are given an integer n. Your task is to determine if this number is a happy number.

A happy number is defined as follows:

  • Start with any positive integer.
  • Replace the number with the sum of the squares of its digits.
  • Repeat the process until the number:
    • Becomes 1 (it will stay there), OR
    • Loops endlessly in a cycle that does not include 1.

If the number eventually becomes 1, it is considered a happy number.

📥 Input Format

  • The first line contains an integer t — the number of test cases.
  • Each of the next t lines contains a single integer n.

📤 Output Format

For each test case, print:

  • Yes if the number is a happy number.
  • No otherwise.

Each output should be on a new line.

🔒 Constraints

  • 1 ≤ t ≤ 30
  • 1 ≤ n ≤ 231 − 1

✅ Sample Input

let n = 29
let res = happyNumber(n)
console.log(res)

📘 Sample Output

Yes

💡 Explanation

  • 19 is a happy number:
    • 1² + 9² = 82
    • 8² + 2² = 68
    • 6² + 8² = 100
    • 1² + 0² + 0² = 1 ✅
  • 2 is not a happy number: it enters a cycle witho
