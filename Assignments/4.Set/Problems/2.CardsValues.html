🃏 Problem: Cards and Values

📖 Description

Chunnu has a collection of cards. Each card has a value assigned to it. He wants to check if there exist two cards N and M such that N is double the value of M (i.e., N = 2 × M).

Print "Yes" without quotes if such a pair exists, otherwise print "No" without quotes.

The values of all the cards are stored in an array arr of length n.

📥 Input Description

The first and only line contains the values of all the cards:

  • The first number is n — the number of cards.
  • The next n space-separated integers represent the values of the cards.

📤 Output Description

Print "Yes" if a solution exists, otherwise print "No". Do not include quotes in the output.

🧪 Sample Input 1


N = 4
cards = [1,2,3,4]
res = checkDoubleExists(cards)
console.log(res)

✅ Sample Output 1

Yes
