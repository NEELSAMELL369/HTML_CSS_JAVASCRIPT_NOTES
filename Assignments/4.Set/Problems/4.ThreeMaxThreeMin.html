🔢 Problem: Three Max, Three Min Please

📝 Description

Given an array of N integers, print the 3 distinct maximum and 3 distinct minimum elements of the array.

📥 Input Description

Input Format:

  • The first line contains an integer N.
  • The second line contains N space-separated integers — elements of the array.

📉 Constraints

  • 0 ≤ N ≤ 100
  • Values in the array can be negative or positive.

📤 Output Description

  • In the first line, print the 3 least distinct values (sorted).
  • In the second line, print the 3 greatest distinct values (sorted).
  • If there are fewer than 3 distinct min or max values, print Not Possible instead (once for each).
  • If both are not possible, print Not Possible twice (once per line).

Note: If there are only 2 distinct min and 2 distinct max values, print Not Possible twice.

🔢 Sample Input 1


const n = 6;
const arr = [1,2,3,4,2,1,6,5];

✅ Sample Output 1

1 2 3
4 5 6

📉 Example Explanation

The 3 smallest distinct values are 1, 2, 3. The 3 largest distinct values are 4, 5, 6. Both conditions are satisfied.
