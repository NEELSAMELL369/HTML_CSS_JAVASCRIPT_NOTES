🔠 Problem: Count Unique Vowels

📖 Description

Write a function that counts the number of unique vowels present in a given string. The comparison should be case-insensitive, meaning 'A' and 'a' are treated the same.

The valid vowels are: a, e, i, o, u.

📥 Input Format

  • A single line containing the input string str.

📤 Output Format

  • A single integer — the count of distinct vowels present in the string.

🔒 Constraints

  • 0 ≤ str.length ≤ 10⁴

🧪 Sample Input 1


let s = "Hello World"
let res = countUniqueVowels(s)
console.log(res);

✅ Sample Output 1

2

💡 Explanation

  • In "Hello World", the vowels are: e, o → total = 2
  • In "education", all vowels a, e, i, o, u are present → total = 5
