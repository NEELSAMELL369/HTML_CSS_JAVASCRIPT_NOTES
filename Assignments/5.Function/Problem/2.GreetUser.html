Title: L0 - Greet User With Prompt Input

Problem Statement:

Write a function expression called greetUser that prompts the user to enter their name using the prompt() function. If the user enters a name, greet them with the message: "Hello, <name>!". If no name is provided (i.e., the user presses "Cancel" or submits an empty string), greet them with: "Hello, Guest!".

Requirements:

  • Use a function expression to define the greetUser function.
  • Handle cases where the user clicks "Cancel" or leaves the input empty.
