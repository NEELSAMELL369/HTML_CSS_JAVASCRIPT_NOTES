🔡 Problem: Longest Substring Without Repeating Characters

📝 Description

You are given a string s consisting of lowercase and/or uppercase English letters, digits, or symbols. Your task is to find the length of the longest substring that contains all unique (non-repeating) characters.

A substring is a contiguous sequence of characters within the string.

📥 Input Format

  • A single line containing the string s.

📤 Output Format

  • Print a single integer — the length of the longest substring without repeating characters.

📊 Constraints

  • 0 ≤ s.length ≤ 104
  • The string s can contain English letters (a–z, A–Z), digits (0–9), and symbols.

🧪 Sample Input 1


            
let str = 'abcabcbb'
let res = longestUniqueSubstring(str)
console.log(res);

✅ Sample Output 1

3

💡 Explanation

The longest substring without repeating characters is "abc", with length 3.
