L1 - Using call() to Access Object Properties

Problem Statement:

Create a function called personInfo that logs the name and age of a person. Using the call() method, call the function with a specific this context (an object with properties name and age).

Example:

const person = {
  name: "John",
  age: 30
};

personInfo.call(person); // Output: Name: John, Age: 30

Solution:

function personInfo() {
  console.log(`Name: ${this.name}, Age: ${this.age}`);
}

const person = {
  name: "John",
  age: 30
};

personInfo.call(person);
