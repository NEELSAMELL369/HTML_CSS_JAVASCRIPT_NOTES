L0 - Transforming and Filtering with map() and forEach()

Problem Statement:

You are given an array of objects representing products. Each product has a name and a price.

Write a function processProducts that:

  • Uses map() to create a new array of product names.
  • Uses forEach() to log each product name along with a message indicating whether the price is above or below $50.

Example:

Input: 
[
  { name: "Laptop", price: 1000 },
  { name: "Mouse", price: 20 }
]
Output (Logged to Console): 
Laptop is above $50
Mouse is below $50
