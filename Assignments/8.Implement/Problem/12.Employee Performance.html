L1 - Employee Performance Evaluation System

Problem Statement:

You are tasked with building an Employee Performance Evaluation System. You have an array of employee objects, where each employee has the following properties: name, tasksCompleted, and rating.

Steps:

  • Filter the employees who have completed more than 5 tasks.
  • Map the filtered employees to a new array that contains only the employee's name and their performance level:
    • If rating is above 4.5 → “Excellent”
    • If rating is between 3 and 4.5 (inclusive) → “Good”
    • Otherwise → “Needs Improvement”
  • Sort the final array in descending order based on performance level: Excellent → Good → Needs Improvement
  • Return the final sorted array containing employee names and their performance levels.

Example:

Input:
[
  { name: "Alice", tasksCompleted: 8, rating: 4.7 },
  { name: "Bob", tasksCompleted: 4, rating: 4.0 },
  { name: "Charlie", tasksCompleted: 6, rating: 3.5 },
  { name: "David", tasksCompleted: 10, rating: 4.9 },
  { name: "Eve", tasksCompleted: 7, rating: 2.8 }
]

Output:
[
  { name: "David", performance: "Excellent" },
  { name: "Alice", performance: "Excellent" },
  { name: "Charlie", performance: "Good" }
]
