Problem Statement:

Write a function deepClone(obj) that creates a deep copy of a given object using JSON.stringify() and JSON.parse(). The cloned object should not affect the original when modified.

Example Input:

{ name: "Alice", hobbies: ["reading", "traveling"] }

Example Output (After modifying the clone):

Original: { name: "Alice", hobbies: ["reading", "traveling"] }
Clone:    { name: "Alice", hobbies: ["reading", "traveling", "coding"] }
