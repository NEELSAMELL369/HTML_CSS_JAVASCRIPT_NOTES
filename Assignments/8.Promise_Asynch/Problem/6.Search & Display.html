You are given an array of task objects. Each task contains the following properties:

  • description: A string describing the task.
  • completed: A boolean indicating whether the task is completed (true) or not (false).

Your job is to:

  1. Sort the tasks by their completion status so that incomplete tasks appear first.
  2. Filter out the tasks that are already completed.
  3. Display the sorted and filtered list of tasks on a web page.

UI Requirements:

  • Add a button labeled "Sort & Filter Tasks" that triggers the logic when clicked.
  • Display each task's description and completion status on the page.
  • Style the task list using cards or boxes for better readability.

Example Input:


const tasks = [
  { description: "Buy groceries", completed: true },
  { description: "Attend stand-up meeting", completed: false },
  { description: "Pay rent", completed: true },
  { description: "Go for a run", completed: false }
];

Expected Output:


Attend stand-up meeting - Incomplete
Go for a run - Incomplete

Constraints:

  • Do not modify the original array.
  • Use DOM methods to render tasks dynamically.
  • Ensure the logic runs only when the button is clicked.
