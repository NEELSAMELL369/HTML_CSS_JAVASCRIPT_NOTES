Title : Creating and Removing Elements

Problem Statement:

Create a simple web page with the following features:

  1. A button labeled "Add Paragraph" that adds a new paragraph to a div each time it is clicked. The paragraph should display the text: "This is a new paragraph."
  2. A button labeled "Remove Last Paragraph" that removes the last paragraph added to the div.

Ensure that the solution uses createElement, appendChild, and removeChild for DOM manipulation.
