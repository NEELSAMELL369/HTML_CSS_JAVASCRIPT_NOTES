Objective:

Create a semantically structured webpage using the following HTML5 elements:

  • <header>
  • <nav>
  • <section>
  • <article>
  • <footer>

Instructions:

  1. In the <header>, add a main title using <h1> and a <nav> with links: Home, About, Contact.
  2. Inside a <section>, create two <article> blocks, each with a <h2> and a <p> describing blog content.
  3. Use a <footer> to display a copyright.

Bonus:

  • Wrap your section inside a <main> tag.
  • Add a <time> tag in each article for a publish date.

Output:

A clean, semantic HTML page structure representing a simple blog layout.
