📄 L0 - Building a Contact Form

📝 Problem Statement:

Design a simple HTML contact form that gathers a user’s information. Use proper form tags to capture the following:

🎯 Form Fields:

  • An <input> field for the user’s name.
  • An <input> field for the email with type="email".
  • A <select> dropdown for gender with options: Male, Female, Other.
  • A <textarea> field for a short message.

📌 Additional Instructions:

  • Label each field clearly using the <label> tag.
  • Include a submit button at the end of the form.

✅ Expected Output:

A simple HTML form structured using semantic and accessible form elements.

💡 Tip: No CSS or JavaScript is required for this task — focus purely on HTML structure.
