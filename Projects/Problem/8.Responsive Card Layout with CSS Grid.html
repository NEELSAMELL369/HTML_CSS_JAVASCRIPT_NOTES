L2 - Create a Responsive Card Layout with CSS Grid

Problem Statement:

Design a fully responsive card layout using CSS Grid. The layout should display cards in a grid format, adjusting the number of columns based on the screen width. For example:

  • Desktop: 4 columns
  • Tablet: 2 columns
  • Mobile: 1 column

Each card should contain an image, a title, and a description.

Requirements:

  • Use display: grid to create the card layout.
  • Define a grid template that adjusts based on the screen size using media queries.
  • Each card should:
    • Have a fixed height and width (adjustable for smaller screens).
    • Include a background color, border, and padding for spacing.
  • Ensure the layout is fully responsive and visually appealing.

Expected Output:

A responsive card layout that adapts seamlessly to various screen sizes.
